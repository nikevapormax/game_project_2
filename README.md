# game_project_2
원래 제작 의도 : 주인공 캐릭터(1)와 적 캐릭터(5)가 있고, 주인공 캐릭터가 주어진 제한시간(1분)동안 적 캐릭터를 피해다니는 것이다. 
                 적 캐릭터는 주인공 캐릭터와 마찬가지로 그냥 걸어다니며 배회하는 것이다. 그리고 시간차를 두고 적 캐릭터가 10초마다 하나씩 늘어나게 하고 싶었다. 


현재 제작 단계 : 주인공 캐릭터와 적 캐릭터 구성은 완벽하게 마쳤다. 특히, 적 캐릭터 구에는 class를 사용했는데, 나머지 기능들에 대해서는 아직 class를 사용해 구현하지 못했다. 
                 코드도 너무 하드코딩처럼 쳐놔서 추후 리팩토링이 필수적이라고 생각한다. 
                 적의 움직임, 캐릭터들의 충돌 부분에서도 class가 된다면 사용하고, 아니라면 함수를 구현해서 코드를 줄여나갈 생각이다.
